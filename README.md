# Hitalk

> 一款快速、简洁且高效的无后端评论系统。  


## 是什么

Hitalk 是一款基于Leancloud的快速、简洁且高效的无后端评论系统。

查看快速开始了解详情。

## 特性

- 快速
- 安全
- 无后端实现
- MarkDown 全语法支持
- 轻量易用(~15kb gzipped)
