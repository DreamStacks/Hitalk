## 是什么

Valine 是一款基于[Leancloud](http://leancloud.cn)的快速、简洁且高效无后端评论系统。

查看[快速开始](quickstart.md)了解详情。

## 特性
- 快速
- 安全
- 无后端实现
- MarkDown 全语法支持
- 轻量易用(~15kb gzipped)
