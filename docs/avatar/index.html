    头像设置¶

    Valine 使用的是Gravatar 作为评论列表头像(感谢gravatar.loli.net提供的镜像服务)。
    目前有以下7种可选值:

    参数值 表现形式 备注
    空字符串'' Gravatar官方图形
    mm 神秘人(一个灰白头像)
    identicon 抽象几何图形(根据邮箱或昵称生成)
    monsterid 小怪物
    wavatar 用不同面孔和背景组合生成的头像(根据邮箱或昵称生成)
    retro 八位像素复古头像(根据邮箱或昵称生成)
    hide   不显示头像

    Info

    参数值hide在版本v1.1.8-beta开始支持。

    如果参数avatar的值不等于列表中的任意一种，则默认为mm。

    new Valine({
    ...
    avatar:'' // (''/mm/identicon/monsterid/wavatar/retro/hide)
});
